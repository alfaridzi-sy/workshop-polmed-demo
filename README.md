# Buku Tamu Digital - Workshop Fundamental GitHub

Repositori ini didedikasikan sebagai ruang praktik langsung (hands-on) kolaborasi kode dalam acara **Workshop Fundamental GitHub** bersama HMPS MI Polmed. Tujuan utama repositori ini adalah mempraktikkan alur kerja Open Source menggunakan metode Fork dan Pull Request.

---

## 📌 Tata Cara Berkontribusi (SOP)

Untuk menjaga kerapian riwayat proyek dan mencegah konflik otorisasi, seluruh peserta wajib mengikuti alur kerja berikut:

1. **Fork Repositori**
   Lakukan *fork* pada repositori ini ke akun GitHub pribadi Anda menggunakan tombol **Fork** di sudut kanan atas.

2. **Clone Repositori**
   Salin repositori hasil *fork* tersebut ke lingkungan lokal (komputer) Anda.
   ```bash
   git clone <URL_FORK_ANDA_DI_GITHUB>
   cd buku-tamu-hmps
